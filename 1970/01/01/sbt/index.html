导入外部jar包

Tips1

非托管依赖像这样工作：将 jar 文件放在 lib 文件夹下，然后它们将会被添加到项目的 classpath 中。没有更多的事情了！

如果用非托管依赖的话，不用往 build.sbt 文件中添加任何内容，不过你可以改变 unmanagedBase key，如果你想用一个不同的目录而非 lib。

sbt package and assembly

sbt-assembly

Building a Thin JAR File

You can build a thin JAR file with the sbt package command. Thin JAR files only include the project’s classes / objects / traits and don’t include any of the project dependencies.

Building a Fat JAR File

myown

You can build fat JAR files by adding sbt-assembly to your project. Fat JAR files inlude all the code from your project and all the code from the dependencies.
